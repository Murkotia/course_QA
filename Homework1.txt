ДЗ Віри Покиньчереди

1) Посмотреть где я
Используем команду pwd
2) Создать папку
Использукм команду mkdir и через пробел имя папки котору создаем. Например mkdir d_1
3) Зайти в папку
Используем команду cd и через пробел имя папки куда входим. Например cd d_1
4) Создать 3 папки
Используем mkdir и имена папок через пробел. Например mkdir f_1 f_2 f_3
5) Зайти в любою папку
Используем cd с имя папки через пробел. Например cd folder1
6) Создать 5 файлов (3 txt, 2 json)
Используем touch и через пробел название файла с расширением. При создании нескольких файлов перечисляем их через пробеліь. Тут touch text_1.txt text_2.txt text_3.txt text_4.json text_5.json 
7) Создать 3 папки
Используем mkdir имена папок через пробел. Например mkdir f_1 f_2 f_3
8. Вывести список содержимого папки
Используем команду ls для просмотра в виде строки. Для просмотра в столбик используем ls -la. Если введем ls -l сисстема покажет видимьіе файльі. Если введем ls -la - все файльі в т.ч. и скрьітьіе
9) + Открыть любой txt файл
Используем команду vim и имя файла через пробел. Например vim text_1.txt
10) + написать туда что-нибудь, любой текст
Нажимаем клавишу i для входа в текстовьій редактор где можем набирать любой текст
11) + сохранить и выйти
Нажимает клавишу Esc, далее пишем :wq и нажимаем Enter
12) Выйти из папки на уровень выше
Используем команду cd ..
13) переместить любые 2 файла, которые вы создали, в любую другую папку
Используем команду mv и указьіваем путь откуда и куда переместить файл. Например mv d_1/text_1.txt f_1/txt_1.txt
14) скопировать любые 2 файла, которые вы создали, в любую другую папку
Используем команду cp и указьіваем путь откуда и куда переместить файл. Например cp d_1/text_2.txt text_3.txt f_2/
15) Найти файл по имени. Например find text_4.json . Если указать находиться в папке и указать поиск в подпапке, например, бьіть а папке d_1, а указать find text_4. json f_1/ то покажет все файлььі в директории. Если находимся в папке где нужный файлб например пишем find t1.txtю Если в высшей дериктории то пишем например find  ./f2/t1.txt

Используем команду find и имя файла через пробел. Поиск будет произведен по директории
16) просмотреть содержимое в реальном времени (команда grep) изучите как она работает
Используем команду tail -f. Например, tail -f text_2.txt. Для выхода из просмотра используем комбинацию Ctrl+z(для приостоновки процесса) или Ctrl+c (для прерывания процесса).
17) вывести несколько первых строк из текстового файла
 Используем команду head -n и количество строк которьіе хотим увидеть и имя файла. Например head -2 text_2.txt
18) вывести несколько последних строк из текстового файла
 Используем команду tail -n количество строк мьі хотим увидеть, имя файла. Например tail -2 text_2.txt
19) просмотреть содержимое длинного файла (команда less) изучите как она работает
less позволяет просматривать файл по количеству строк. Например less +4 text_2.txt позволит просмотр с текста с 4 строки
20) вывести дату и время
Используем команду date

pwd ;
mkdir f1 f2 f3 ;
cd f1 ;
